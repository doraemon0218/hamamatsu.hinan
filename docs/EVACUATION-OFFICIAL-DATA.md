# 指定避難場所フラグの根拠（行政公開データ）

訓練画面のトップ5で表示する「指定避難場所」かどうかは、次の**公開データ・公式ページ**と施設名を突き合わせて設定しています。データが更新されたら、同じ手順で再照合してください。

## 浜松市（`region=hamamatsu` 既定）

| 根拠 | URL |
|------|-----|
| 静岡県オープンデータ「指定緊急避難場所一覧」（浜松市・221309_evacuation_space.csv） | https://opendata.pref.shizuoka.jp/dataset/10983.html |
| 浜松市「津波避難ビルの指定について」（HTML 一覧・PDF） | https://www.city.hamamatsu.shizuoka.jp/kiki/disaster/bousai/building/ |

- **指定あり（true）**: CSV の「名称」に一致するもの（例：浜松城公園、浜松市役所、東小学校、浜松中部学園（小・中学校）、双葉小学校、西小学校）。
- **指定なし（false）**: 当該 CSV の名称に該当がない施設（浜松アクトタワー、浜松駅前ビル）。市サイト「津波避難ビル一覧」HTML 掲載分でも当該名称は確認できず（PDF は未機械照合）。

## 串本町（`region=kushimoto`）

| 根拠 | URL |
|------|-----|
| BODiK「【和歌山県】避難先情報一覧」 | https://data.bodik.jp/dataset/300004_w090200_hinanjo |
| 串本町 CSV（上記データセット内） | データセットページから「避難先情報一覧 串本町CSV」 |
| 串本町「避難場所一覧」 | https://www.town.kushimoto.wakayama.jp/bousai/hinan-basho.html |

- **指定あり（true）**: 例）県営住宅串本団地(3階以上)、サンゴ台集会所、串本中学校（体育館）等と同一施設と判断できるもの、串本古座高等学校（一覧の「串本古座高校」系施設）。
- **指定なし（false）**: 東牟婁振興局串本建設部（一覧に当該名称なし）、串本橋北詰付近（アプリ独自の地点ラベルで一覧施設名と不一致）。

## 大阪・梅田（`region=umeda`）

| 根拠 | URL |
|------|-----|
| 大阪市北区「津波避難ビルの一覧【民間施設】」 | https://www.city.osaka.lg.jp/kita/page/0000299035.html |
| 大阪市「指定緊急避難場所・指定避難所の指定について」 | https://www.city.osaka.lg.jp/kikikanrishitsu/page/0000349214.html |

- **指定あり（true）**: 北区民間津波避難ビル一覧の「グランフロント大阪 南館、北館」に、グランフロント大阪（北館）が含まれるため。
- **指定なし（false）**: 当該民間施設一覧の表に、大阪駅・梅田スカイビル・阪急百貨店・梅田ツインタワーズ・ヒルトン大阪・「周辺ビル」としての記載なし（他区分の指定は PDF 等で要確認）。

## 注意

- 災害の種類（津波・地震等）ごとに指定が異なる場合があります。実際の避難は**各自治体のハザードマップ・最新の公示**を優先してください。
- 本アプリの地点はデモ用の名称・座標を含むため、公式データの表記揺れ・校舎単位の行がある場合は人手で同一施設か判断しています。
